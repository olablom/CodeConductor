# Security Audit for Banking Application

Perform a comprehensive security audit of a banking application with the following requirements:

- Authentication and authorization systems
- Data encryption and protection
- API security and rate limiting
- Database security and access controls
- Compliance with financial regulations (PCI DSS, GDPR)
- Penetration testing recommendations
- Security monitoring and logging
- Incident response procedures

The application handles sensitive financial data and must meet enterprise-grade security standards.
